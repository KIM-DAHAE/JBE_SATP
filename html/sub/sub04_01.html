고입전형 맞춤형 가이드

알맞은 곳에 체크를 한 후
맞춤 가이드를 확인해보세요.

  • 1. 졸업예정인 중학교가 어느 지역입니까?

  • 2. 전라북도내 진학하고자 하는 곳은 어디입니까?

  • 3. 해당지역의 진학하고자 하는 학교는 무슨 계열입니까?

고입전형 맞춤형 가이드

  • 입력정보가 맞습니까?

    • 출신학교 구분   |   전라북도내 지역 중학교
    • 진학희망 지역   |   평준화지역(전주, 군산, 익산)
    • 진학희망 계열   |   일반고
고입전형 맞춤형 가이드

일반고 ( 평준화 )

  • 1. 진학상담

    본인의 소질 및 적성, 성적 등을 고려하여 진로결정을 합니다.

  • 2. 원서작성

    평준화일반고등학교는 담임선생님을 통하여 NEIS(입학)에서 원서를 작성 및 전송합니다.

    • 지망정보는 배정시 사용되는 중요한 정보이므로 정확히 입력되었는지 확인해야 한다.

  • 3. 원서접수

    작성된 고등학교 입학 온라인 원서는 시험지구(전주,군산,익산)교육지원청으로 전송되며 원서가 접수되는 순서대로 접수번호가 부여됩니다.

고입전형 맞춤형 가이드

일반고 ( 평준화 )

  • 4. 수험번호부여

    원서접수가 마감되면 교육청에서 수험번호를 부여합니다.

고입전형 맞춤형 가이드

일반고 ( 평준화 )

  • 5. 합격자발표

    원서접수가 마감되면 교육청에서 수험번호를 부여합니다.

    • 지망정보는 배정시 사용되는 중요한 정보이므로 정확히 입력되었는지 확인해야 한다.

  • 6. 고등학교 배정

    교육청에서 평준화 지역 합격자를 대상으로 선배정, 일반배정을 통하여
    입학 고등학교를 배정합니다.

고입전형 맞춤형 가이드

일반고 ( 평준화 )

  • 7. 배정예시

    일반지망 배정

    • 1지망 모집정원이 30명인 A 고교 / 1지망 모집 정원이 30명인 B 고교 /
      1지망 모집정원이 30명인 C 고교
    • 30명 초과시 고입포털 시스템 무작위 추첨
